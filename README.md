# demo-bookshop-service

The demo-bookshop-service is a Spring Boot application that provides a RESTful API for managing a bookshop catalog. 
It supports operations such as listing books, retrieving book details, and updating book information. 
The project uses Java, Gradle, and integrates Spring REST Docs for API documentation.

## Prerequisites

Testing and running the application requires:
- Java 11 or higher
- Gradle 6.0 or higher
- An IDE such as IntelliJ IDEA or Eclipse (optional)
- cURL or Postman for testing the API endpoints
- Git for version control (optional)
- Docker (optional, for containerization)
- A web browser for accessing the API documentation
